# simulator-game
A game for simulating manufacturer 

## Pre-requisites 
Before you start develop and run this web application, please make sure you have installed the following tools on your local environment

- Node and NPM 
- Yarn
- Webpack
- JDK 12 (You can download oracle jdk 12 or openjdk 12 )
- Gradle 5.x
- MySQL Server
- Docker (optional)
